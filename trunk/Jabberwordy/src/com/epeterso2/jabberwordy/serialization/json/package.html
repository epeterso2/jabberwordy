A collection of classes for reading and writing puzzles in the JavaScript Object Notation (JSON) format.

The JSON puzzle format is a file format developed by Jim Horne. The format is based upon the JSON format and was designed to provide flexibility in manipulating crossword puzzles in web browsers. The JSON Specification is available on XWordInfo.

A single JSON puzzle is represented by a {@link com.epeterso2.jabberwordy.serialization.json.JSONPuzzle} object.

  • The {@link com.epeterso2.jabberwordy.serialization.json.JSONPuzzleOutputStream} is used to convert a byte[] representation of a JSON puzzle file into a {@link com.epeterso2.jabberwordy.serialization.json.JSONPuzzle} object.
  • The {@link com.epeterso2.jabberwordy.serialization.json.JSONPuzzleInputStream} is used to convert a {@link com.epeterso2.jabberwordy.serialization.json.JSONPuzzle} object into a byte[] representation of a JSON puzzle file.
